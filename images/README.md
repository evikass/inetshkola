# 📁 Структура папок изображений ИНЕТШКОЛА

Эта папка содержит все изображения для образовательной платформы.

## 📂 Структура

```
/images/
├── classes/              # Изображения для классов (обложки, баннеры)
│   ├── class-0.jpg       # Подготовительный класс
│   ├── class-1.jpg       # 1 класс
│   ├── class-2.jpg       # 2 класс
│   └── ...
│
├── subjects/             # Изображения по предметам
│   ├── math/             # Математика
│   │   ├── hero.jpg      # Главное изображение
│   │   ├── icons/        # Иконки тем
│   │   └── illustrations/# Иллюстрации к урокам
│   ├── russian/          # Русский язык
│   ├── literature/       # Литература
│   ├── english/          # Английский язык
│   ├── science/          # Окружающий мир
│   ├── history/          # История
│   ├── geography/        # География
│   ├── biology/          # Биология
│   ├── physics/          # Физика
│   ├── chemistry/        # Химия
│   ├── informatics/      # Информатика
│   ├── art/              # ИЗО
│   ├── music/            # Музыка
│   ├── pe/               # Физкультура
│   ├── obzh/             # ОБЖ
│   └── social/           # Обществознание
│
├── sections/             # Изображения для разделов
│   └── section-id/       # По ID раздела
│
├── topics/               # Изображения для тем
│   └── topic-id/         # По ID темы
│
├── lessons/              # Изображения для уроков
│   └── lesson-id/        # По ID урока
│
├── icons/                # Общие иконки
│   ├── subjects/         # Иконки предметов
│   ├── actions/          # Иконки действий
│   └── ui/               # UI иконки
│
├── backgrounds/          # Фоны
│   ├── hero/             # Главные баннеры
│   ├── cards/            # Фоны карточек
│   └── patterns/         # Паттерны
│
├── illustrations/        # Общие иллюстрации
│   ├── characters/       # Персонажи-помощники
│   ├── achievements/     # Иллюстрации достижений
│   └── education/        # Образовательные иллюстрации
│
├── achievements/         # Значки достижений
│   ├── first-step.svg
│   ├── explorer.svg
│   └── ...
│
└── badges/               # Значки и ранги
    ├── novice.svg
    ├── student.svg
    └── ...
```

## 📏 Рекомендуемые размеры

| Тип | Размер | Формат |
|-----|--------|--------|
| Обложка класса | 1920x600 | WebP/JPG |
| Главное изображение предмета | 800x600 | WebP/JPG |
| Иконка предмета | 128x128 | SVG/PNG |
| Иллюстрация урока | 1200x800 | WebP/JPG |
| Иконка достижения | 256x256 | SVG |
| Фон карточки | 400x300 | WebP/PNG |
| Иконка действия | 48x48 | SVG |

## 🎨 Стиль изображений

- **Цветовая палитра**: Тёмная тема (slate-900, slate-800)
- **Акцентные цвета**: blue-500, purple-500, green-500
- **Стиль**: Современный, плоский, с градиентами
- **Формат**: Предпочтительно WebP для оптимизации

## 📝 Именование файлов

- Используйте kebab-case: `math-fractions.jpg`
- Добавляйте суффиксы для размеров: `hero-sm.jpg`, `hero-lg.jpg`
- Для иконок используйте SVG: `icon-calculator.svg`
